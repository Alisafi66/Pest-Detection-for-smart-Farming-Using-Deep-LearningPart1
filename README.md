# Pest Detection for Smart Farming Using Deep Learning

This repository contains the implementation and evaluation of my Final Year Project (FYP) titled **"Pest Detection for Smart Farming Using Deep Learning"** at Multimedia University, Malaysia.

The project explores the use of **zero-shot object detection** using two AI models:
- **Grounding DINO (Model 1)**
- **AIM-V2 (Model 2)**

The objective is to identify pests from images with high accuracy and recommend the most suitable model based on evaluation and expert farmer feedback.
